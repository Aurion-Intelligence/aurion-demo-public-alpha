# Contributing to Aurion (Demo Public Alpha)

Thanks for your interest in Aurion. This is an early, governance-first **Demo Public Alpha**, open
source under **GNU AGPL-3.0-only** (see [`LICENSE`](LICENSE)).

## Feedback and issues — welcome now

- **Bug reports, questions, and feedback are welcome.** Open an issue describing what you tried, what you
  expected, and what you saw.
- If something looks "fake-green" (claims more readiness than it proves), please say so — honest
  readiness is a design goal.
- Discussion, reproductions, and design suggestions are all helpful at this stage.

## Code contributions — temporarily paused

**External code contributions (pull requests) are not being merged yet.**

Before we can accept and redistribute external code under AGPL-3.0-only — and keep the option of any
future alternative licensing — we need a clear **contributor-rights / CLA process** in place. That
process is **not finalized**, so we are **not merging external code changes** for now.

- Please **do not** invest significant effort in large pull requests yet; we may not be able to merge
  them until the contributor-rights process is finalized.
- Small typo/doc fixes may still be filed as issues so we can track them.
- When the contributor-rights process is ready, this document will be updated with how to sign and
  submit contributions.

## Licensing notes

- The project is licensed **AGPL-3.0-only**. Contributions discussed here would be under AGPL-3.0-only
  once the contributor process is finalized.
- **No paid commercial license exists today.** An alternative commercial license may be developed later,
  **subject to legal review** — it is not offered now.
- The **plugin / limb licensing boundary is still under review.** We do not currently promise that every
  independent plugin may remain proprietary; whether a plugin is an AGPL derivative work is
  fact-specific and not yet settled for Aurion.
- Advanced/private Aurion modules are intentionally **not** part of this public repository.

## What this repository is (and is not)

This is a bounded demo of Aurion's governed mission spine. It is **not** production-ready, **not** a full
public alpha, and **not** a claim of complete unattended autonomy. See
[`docs/alpha/KNOWN_LIMITATIONS_PUBLIC.md`](docs/alpha/KNOWN_LIMITATIONS_PUBLIC.md).
